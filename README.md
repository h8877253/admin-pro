# Admin Pro

开箱即用的中台前端/设计解决方案。

![](https://gw.alipayobjects.com/zos/rmsportal/xEdBqwSzvoSapmnSnYjU.png)

## 使用

```bash
$ git clone https://github.com/Mulgore/admin-pro
$ cd admin-pro
$ npm install
$ npm start         # 访问 http://localhost:8000
```

## 兼容性

现代浏览器及 IE11。
